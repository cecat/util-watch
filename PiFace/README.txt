This code works once you have set up and updated Raspbian per instructions here:
http://www.element14.com/community/thread/39390/l/piface-digital-2--setup-and-use

In particular, with an updated Raspbian system I installed the piface software per:
wget http://pi.cs.man.ac.uk/download/install.txt | bash
sudo apt-get install python3-pifacedigitalio python-pifacedigitalio
sudo apt-get install python3-pifacedigital-emulator
sudo apt-get install python3-pifacedigital-scratch-handler

then rebooted.

The code is based on the PiFace Digital I/O Documentation, Release 3.1.0
(Thomas Preston, December 25, 2015)

https://media.readthedocs.org/pdf/pifacedigitalio/latest/pifacedigitalio.pdf
